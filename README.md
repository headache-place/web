# [두통과 함께하는 사람들](https://headache.place)

카페 "두통과 함께하는 사람들"의 검색 엔진 최적화용 웹을 구성하는 Next.js 프로젝트입니다.

## Technologies

- Node.js v20, pnpm
- Vercel for Deployment
- Next.js App Router + Tailwind CSS
- ContentLayer for MDX.
